# 参数和公式说明
## 参数
### Player
#### 技术熟练度(0~100)  
| 参数    | 名称 |
|-------|----|
| serve | 发球 |
| pass  | 垫球 |
| set   | 托球 |
| spike | 扣球 |
| block | 拦网 |

#### 基本能力(0~100)
| 参数               | 名称   | 含义         |
|------------------|------|------------|
| strength         | 力量   | 力量         |
| height           | 高度   | 弹跳高度、击球点高度 |
| flexibility      | 灵活   | 反应速度、手法等   |
| reliable         | 稳定   | 表现的稳定程度    |
| stamina_resist   | 体力抵抗 | 抵抗体力下降的能力  |
| mentality_resist | 心态抵抗 | 抵抗心态下降的能力  |

#### 状态(0~100, 初始为100)
| 参数        | 名称 |
|-----------|----|
| stamina   | 体力 |
| mentality | 心态 |

### Ball
| 参数             | 名称  | 含义                        |
|----------------|-----|---------------------------|
| power          | 威力  | 速度、力量                     |
| disruptiveness | 破坏性 | 运动复杂程度、处理难度               |
| distance       | 距离  | 球在场上的位置(0~9, 超出则判定为过网或出界) |

## 公式
1. 调用数值时，采用正态分布模拟水平波动，标准差为  
    $$\frac{10}{\frac{100}{稳定}* \frac{100}{体力}* \frac{100}{心态}}$$

2. 需要判定时，一般使用以下方法组合比较：  
    $对应参数的加权和$  
    $能力因子-挑战因子$  
    $100-20*e^{\frac{x}{10}}$  
    $1+\frac{能力因子}{常数}$  
    $\frac{能力因子^{2}}{能力因子^{2}+挑战因子^{2}}$  
    $设定范围的随机数$  
    $设定均值及标准差的正态分布随机数$  

3. 需要计算时，一般使用以下方法进行组合求值：  
    $对应参数的加权和$  
    $能力因子-挑战因子$  
    $(常数-挑战因子)*(能力因子)$  
    $(常数-控制因子)*挑战因子+(常数+控制因子)*能力因子$  
    $设定均值及标准差的正态分布随机数$
